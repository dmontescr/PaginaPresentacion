# Portfolio Express

Bienvenido a **Portfolio Express**, un proyecto de página web personal sencilla y moderna creada con HTML, CSS y JavaScript puro.

## 🚀 Características

- **Diseño Moderno y Limpio**: Interfaz de usuario profesional, atractiva y con un aspecto *premium*.
- **Totalmente Responsivo**: Adaptado para visualizarse correctamente tanto en ordenadores de escritorio como en dispositivos móviles.
- **Secciones Integradas**:
  - **Hero**: Presentación principal con título llamativo y llamada a la acción.
  - **Sobre mí**: Breve biografía, misión y objetivos.
  - **Habilidades**: Tarjetas informativas con iconos sobre competencias técnicas (Inteligencia Artificial, Automatizaciones, Creación de apps, Formación online).
  - **Proyectos**: Ejemplos de proyectos destacados utilizando un diseño elegante con gradientes en CSS.
  - **Contacto**: Sección final interactiva para recibir propuestas o mensajes.
- **Interactividad**: Scroll suave (*smooth scroll*) para la navegación interna del menú y animaciones fluidas (*fade-in* y desplazamiento) en las tarjetas a medida que bajas por la página.

## 🛠️ Tecnologías utilizadas

- **HTML5**: Estructura semántica de la web.
- **CSS3**: Estilos personalizados, uso intensivo de Flexbox y CSS Grid, variables CSS (Custom Properties) para un mantenimiento sencillo y animaciones.
- **JavaScript (Vanilla)**: Interactividad en el cliente (gestión del scroll suave, eventos de clic y el uso de `IntersectionObserver` para las animaciones al hacer scroll).

## 📂 Estructura del Proyecto

```text
PaginaPresentacion/
├── index.html       # Estructura principal de la página
├── style.css        # Hoja de estilos principal
├── script.js        # Lógica de interactividad y animaciones
└── README.md        # Documentación del proyecto (este archivo)
```

## 💻 Cómo ejecutar el proyecto

Al ser un proyecto estático básico de *frontend*, no necesitas instalar ninguna dependencia ni configurar un entorno especial de servidor. Simplemente:

1. Navega hasta la carpeta del proyecto en tu ordenador.
2. Abre el archivo `index.html` con tu navegador web favorito (Google Chrome, Firefox, Safari, Edge, etc.).
3. ¡Disfruta explorando el portfolio!
